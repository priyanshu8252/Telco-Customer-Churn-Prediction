# Telecom Customer Churn Prediction & Analytics

## 📌 Project Overview
This project aims to help a telecommunications company identify customers who are at high risk of canceling their subscriptions (churning). By applying predictive modeling and data analytics, the goal is to uncover behavioral patterns that lead to attrition and provide actionable insights for customer retention strategies.

## 🛠️ Features & Methodology
- **Data Preprocessing & Cleaning:** Handled missing data types, resolved class imbalances, and handled categorical encoding.
- **Exploratory Data Analysis (EDA):** Visualized trends between demographics, tenure, contract types, and monthly fees against customer churn rates.
- **Machine Learning Classifiers:** Implemented and evaluated predictive models including Logistic Regression, Support Vector Machines (SVM), and Random Forest.
- **Evaluation Metrics:** Evaluated models using Classification Reports, Confusion Matrices, and ROC-AUC curves.

## 📁 Repository Structure
- `Telco Customer Churn Prediction.ipynb` - The primary Jupyter Notebook containing data pipelines, charts, and machine learning code.
- `Telco-Customer-Churn.csv` - The customer dataset containing demographic and billing features.
